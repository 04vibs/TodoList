API SERVER RUNS ON PORT 9000.
MODELS: The database model contains of one table todos.
API: api contains of index.js file which accepts the request coming to the route
https://localhost:9000/todos
Testing of the project is done using Postman application.
Accepts the following requests:
1) /GET: returns all the currently availabel todos in the database,so that it can be printed .
2) /POST:Add the items in the todos table.
